## ✨ Features

- **🌈 Dynamic Browsing:** Browse over 100+ community-created `.attheme` files within the app.
- **🖼️ Real-time Previews:** High-fidelity bubble previews for background, incoming, and outgoing messages.
- **⚡ One-Tap Installation:** Instant download and seamless application with official persistence.
- **🌥️ Remote Repository:** Always stay updated with new themes without ever needing to update the app.

## 🚀 How it Works

The Alexgram Theme engine utilizes a sophisticated `themes.json` manifest to index remote assets. Our custom activity fetches this manifest, renders a GPU-accelerated grid of previews, and utilizes official Telegram integration to ensure your selected theme stays active even after app updates.
